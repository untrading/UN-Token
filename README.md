# untrading UN Token Contracts

https://untrading.org

This repo contains the ERC20 UN Token and accompanying vesting contract.

## Development

Install dependencies with Forge:

```bash
forge install
```

Compile the contracts with:

```bash
forge build
```

## Testing

Test contracts with Forge:

```bash
forge test
```
